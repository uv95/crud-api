# CRUD-API

## To launch:
 - Add ```.env``` file with ```PORT``` variable;
 - Run ```npm i```;
 - Run ```npm run start:dev``` to start in dev mode;
 - Run ```npm run start:prod``` to build and run the bundled file;
 - Run ```npm run start:multi``` to start multiple instances.

## To test:
 - Run ```npm run test```.
